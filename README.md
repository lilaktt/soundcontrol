# 🎧 Sound Control (v1.21.4 - 1.21.8)

![Fabric](https://img.shields.io/badge/Modloader-Fabric-lightgrey?style=flat-square)
![Minecraft Versions](https://img.shields.io/badge/Minecraft-1.21.4%20--%201.21.8-brightgreen?style=flat-square)

**Sound Control** is a client-side Fabric utility mod that gives you complete, granular control over every single sound in Minecraft. 

## ✨ Features

* ⭐ **Favorites System:** Pin your most frequently tweaked sounds to the top of the list for quick and easy access using the **(★)** button.
* 🎚️ **Individual Volume Control:** Adjust the volume or completely mute *any* specific sound in the game independently.
* 🔍 **Smart Search & Filters:** Quickly find sounds using the search bar, or filter by categories (Mobs, Blocks, All) and Status (Edited, Favorites).
* 👀 **Live Sound Tracker:** Toggle an on-screen overlay to see exactly which sound IDs are currently playing around you in real-time.
* 🌍 **Client-Side Only:** No server-side installation required. Works on any vanilla or modded server.
* 🌐 **Fully Localized:** Supports 12+ languages including English, Ukrainian, German, French, and more.

## 🎮 How to Use

* **`V`** - Open the main Sound Control GUI.
* **`Y`** - Toggle the Live Sound Tracker overlay on/off.
* In the menu, simply click the **Play (▶)** button to preview, the **Mute** button to silence, or the **Star (★)** to add to Favorites.

## 📦 Installation

1. Install the [Fabric Loader](https://fabricmc.net/).
2. Download the version for your Minecraft version (1.21.4+) and place it in your `mods` folder.
3. Launch the game!
